# 临床医生智能排班系统 (Clinical Doctor Intelligent Scheduling System)

**临床医生智能排班系统** 是一个专为医院临床科室量身定制的现代化排班与人员管理解决方案。本系统旨在将繁琐、复杂的排班工作数字化、自动化，帮助科室管理者高效地进行医护人员调度、班次分配及考勤记录，同时支持多科室隔离、细粒度的权限控制以及灵活的模块化规则。

---

## 🌟 核心功能

- **🤖 智能排班管理**：支持自定义班次、灵活排班、日历视图展现，自动识别节假日与特殊排班规则。
- **🏥 多科室支持与数据隔离**：支持同一系统内管理多个独立科室，各科室数据完全隔离，支持复制科室配置以快速初始化。
- **👥 灵活的人员与角色配置**：
  - 支持对科室人员进行动态管理。
  - 内置三种安全角色：
    - `admin` (管理员)：可编辑人员、排班及科室配置。
    - `terminal` (终端机)：主要用于科室大屏或固定终端展示排班情况，拥有特定操作权限。
    - `guest` (访客)：仅支持查看排班数据。
- **🧩 高级模块化设计**：排班表支持模块化动态挂载，可根据各科室需求定制排班显示规则、统计模式与颜色高亮。
- **📢 公告与历史记录管理**：内置科室公告（如教学门诊通知、特殊提示），并支持公告历史版本追溯。
- **📄 数据导出**：支持一键将排班表导出为 Excel (`.xlsx`) 或 PDF 文件，方便打印与存档。
- **✅ 自动化验证**：内置完整的 E2E（端到端）自动化回归测试套件，保障系统在升级时的业务稳定性。

## 💻 技术栈

### 后端 (Backend)
- **Node.js**
- **Express.js**：提供轻量级的 RESTful API 服务。
- **JSON 文件存储**：轻量化部署，无需额外配置数据库即可运行，支持数据自动备份。

### 前端 (Frontend)
- **HTML5 + Vanilla JS / Vue 3** (全局引入模式)
- **Tailwind CSS**：现代化的原子级 CSS 框架。
- **Quill Editor**：轻量级富文本编辑器，用于公告发布。
- **XLSX & html2pdf.js**：前端生成报表并导出 Excel / PDF。

### 测试与工具 (Testing & Tooling)
- **Puppeteer & JSDOM**：完整的端到端自动化测试与 UI 验证。

## 🚀 安装与启动

本项目无需繁杂的环境配置，确保已安装 [Node.js](https://nodejs.org/) (建议 v18+) 即可快速运行：

1. **克隆项目 / 下载源码**
   ```bash
   git clone https://github.com/cassiuschen9261-cmd/-.git
   cd -
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动服务**
   - **通过 npm 启动：**
     ```bash
     npm start
     ```
   - **通过脚本一键启动 (Windows)：**
     双击根目录下的 `start_server.bat` 或运行 `start_server_cn.ps1`。
   
   服务启动后，在浏览器中访问：`http://localhost:8000` (默认端口可能因配置有所不同，请留意终端输出)。

## 🧪 自动化测试

系统内置了全面的端到端测试套件，用来验证模块侧边栏、导出权限、UI 布局、部门管理等多项核心功能：

- **运行全量测试**：
  ```bash
  npm test
  ```
- **生成测试报告**：
  ```bash
  npm run report
  ```
- **清理测试产物**：
  ```bash
  npm run cleanup
  ```
